linux 解压 tar 文件#

要在Linux上解压缩.tar文件，可以使用以下命令：

tar -xvf file.tar

这将解压缩.tar文件并将其提取到当前目录中。您可以将“file.tar”替换为您要解压缩的文件的实际名称。

此外，您可以使用以下选项来指定解压缩的位置和/或将解压缩的文件重定向到另一个文件：

  • -C：指定要解压缩的目标目录。例如，要将文件提取到“/home/user/files/”目录中，请使用以下命令：

    arduino

    1

    tar -xvf file.tar -C /home/user/files/

  • -f：指定要解压缩的文件名。这个选项必须出现在命令行的最后一个。

  • -O：将解压缩的文件重定向到标准输出。例如，要将文件提取并打印到屏幕上，请使用以下命令：

    mathematica

    1

    tar -xvf file.tar -O

请注意，在解压缩.tar.gz文件时，您可以使用“z”选项来启用gzip压缩：

tar -xzvf file.tar.gz

同样，您可以使用“j”选项来启用bzip2压缩：

tar -xjvf file.tar.bz2

ubuntu 安装nvm#

要在Ubuntu上安装nvm（Node Version Manager），可以按照以下步骤操作：

  1. 首先，使用curl命令下载nvm安装脚本。您可以使用以下命令：

    bash

    1

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

    上述命令会下载nvm安装脚本并在bash shell中运行。

  2. 安装脚本运行后，需要注销当前的终端并重新登录才能使nvm生效。您也可以运行以下命令来使nvm生效：

    bash

    1

    source ~/.bashrc

    这将重新加载bash shell并使nvm可用。

  3. 安装完成后，您可以使用以下命令来安装Node.js：

    nvm install node

    这将安装最新版本的Node.js。您还可以安装指定版本的Node.js，例如：

    nvm install 14.17.6

    这将安装Node.js的14.17.6版本。

  4. 使用以下命令来验证Node.js是否已正确安装：

    node -v

    这将显示已安装的Node.js版本号。

注意：上述步骤假设您正在使用bash shell。如果您使用的是其他shell，请相应地更改命令。

ubuntu安装pm2#

要在Ubuntu上安装pm2，可以按照以下步骤操作：

  1. 首先，请确保您已经安装了Node.js和npm。您可以使用以下命令来检查它们是否已安装：

    node -v npm -v

    如果它们没有安装，请按照适当的方式安装它们。

  2. 安装pm2，可以使用npm包管理器来完成。在终端中运行以下命令：

    npm install pm2 -g

    这将使用npm全局安装pm2。

  3. 安装完成后，您可以使用以下命令来验证是否已正确安装pm2：

    css

    1

    pm2 --version

    这将显示已安装的pm2版本号。

  4. 现在，您可以使用pm2启动Node.js应用程序。例如，要使用pm2启动名为myapp.js的应用程序，请使用以下命令：

    sql

    1

    pm2 start myapp.js

    这将启动应用程序并使用pm2管理它。您可以使用以下命令来查看所有正在运行的应用程序：

    pm2 list

    您可以使用其他命令，如pm2 restart、pm2 stop和pm2 delete来管理应用程序。

注意：上述步骤假设您已经正确设置了Node.js应用程序并已准备好使用pm2来管理它们。如果您需要设置应用程序，请按照适当的方式完成。

ubuntu中使用pm2启动node项目#

要在Ubuntu中使用pm2启动Node.js项目，您可以按照以下步骤操作：

  1. 首先，请确保您已经安装了Node.js和npm。您可以使用以下命令来检查它们是否已安装：

    node -v npm -v

    如果它们没有安装，请按照适当的方式安装它们。

  2. 在您的Node.js项目目录中，使用以下命令安装所有依赖项：

    npm install

    这将安装项目所需的所有npm包。

  3. 安装pm2，可以使用npm包管理器来完成。在终端中运行以下命令：

    npm install pm2 -g

    这将使用npm全局安装pm2。

  4. 现在，您可以使用pm2启动Node.js应用程序。例如，要使用pm2启动名为bin/www的应用程序，请使用以下命令：

    1

    pm2 start bin/www

    这将启动应用程序并使用pm2管理它。您可以使用以下命令来查看所有正在运行的应用程序：

    pm2 list

    您可以使用其他命令，如pm2 restart、pm2 stop和pm2 delete来管理应用程序。

  5. 如果您需要配置pm2以在系统启动时自动启动应用程序，请使用以下命令：

    pm2 startup systemd

    这将创建一个systemd服务，以在系统启动时启动pm2并启动您的应用程序。执行此命令后，pm2将为您提供一条命令，您需要复制并在终端中运行它以完成服务的创建。

注意：上述步骤假设您已经正确设置了Node.js应用程序并已准备好使用pm2来管理它们。如果您需要设置应用程序，请按照适当的方式完成。
